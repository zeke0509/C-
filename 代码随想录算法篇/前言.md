# 时间复杂度
## 解题技巧
1. 可以列个表
   e.g1. 下列函数的时间复杂度是
	```
	int fun(int n) {
		int i = 0, sum = 0;
		while(sum<n) sum += ++i;
		return i;
	}
	```
	\==\==解答过程\==\===
	次数|   1   |   2    |    3   |    4   |    5    |    t    |
	i      |   1   |   2    |    3   |    4   |    5    |    t    |
	sum|   1   |  1+2 |1+2+3|
	得到sum = (1+t)t/2
	当sum >= n 跳出循环：利用取大头，则为t^2 >= n，则t = n^1/2 
	所以时间复杂度为：O(n^1/2)

	外层次数        |     1     |      2      |      3      |      4     |     t      |
	   i               |  2^0    |   2^1    |   2^2     |   2^3   |2^(t-1) |

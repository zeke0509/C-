1. 处理不确定数量的输入可以用：```while(std::cin >> a >> b)``` 来判断
2. 对于多行输入，想要读取一整行内容
   - 行内有空格，如
     3
     hello world
     使用 getline() 获取
		```
		int n;
		string s1;
		cin >> n;
		cin.get();        // 吸收换行符（关键）
		getline(cin, s1);
		```
   - 行内无空格，如
     1
     aabbcc
     使用cin即可
		```
		int n;
		string s1;
		cin >> n;
		cin >> s1;
		```

# 一些做题技巧
- 反转字符串的题目，不要直接用reverse函数，但是可以用swap函数
  swap(&x, &y); 传入两个元素的引用并交换。
- 替换字符串：
  先扩容，再从尾部赋值。
- 去掉字符串中冗余空格（即去除：首、尾空格，每个单词间只留一个空格）
  思路：
  维护两个指针，一个快指针用于遍历找到单词首字符，一个慢指针用于赋值
  以单词为单位进行处理
  具体做法：
  维护两个指针，一个快指针用于遍历，一个慢指针用于重新赋值
  遍历字符串，当遇见非空字符（即单词的首字母）开始处理：
	  （排除首单词）让慢指针=‘ ’ ，并让慢指针+1
	  通过while循环，让每个慢指针=快指针，直至这个单词末尾
  当遇见空字符不处理即不给慢指针赋值：
  代码实现
	```
	int slowI = 0;
	for (int i = 0; i < s.size(); i++) {
		if (s[i] != ' ') {
			if (slowI) s[slowI++] = ' ';
			while (s[i] != ' ' && i < s.size()) s[slowI++] = s[i++];
		}
	}
	s.resize(slowI);
	```
- KMP算法
  next数组中：记录的是全局对称性，例如```next[5] = 2; // 说明开头有两个字符s[0]、s[1]是和末尾5重叠的；这样就说明对于s[6]直接去找s[next[5]]比较即可。```再进一步理解：next数组存放的是真实重叠字符个数，那么它会比索引末端大1。但是我要比较的字符串正好是6（=5+1），正好利用了数组存放值比索引大1的性质抵消掉了。
  next数组的本质：就是不断比较当前字符与```s[next[j]], 其中j为当前字符的前一个字符对应的next数组值```
- 如何判断重复子字符串？
  利用KMP方法的next数组。如果```(len - next[len-1])可以整除len，且next[len-1]>0，其中len为原字符串长度```，说明原字符串由一个大小为```len - next[len - 1]的循环结构```

# 一些做题技巧
- 在集合中查找并只保留一个（不在乎集合是否被更改）的一种优雅写法：
  ```if (mySet.erase(x))```
  替代find()方法。
- 数组求和是用哈希法还是双指针法？
  题目中有多个数组：哈希法，可以通过哈希查找，来查找别的数组的值
  题目中只有一个数组：双指针法，在同一个数组中进行遍历。

# 其它优化
- ```int cnt[26]{};  替代 int cnt[26] = {0};```
- ```count[ch - 'a']--;   if (count[ch - 'a'] < 0) return false;```
  可以简化为：```if (--count[ch - 'a'] < 0) return false;```

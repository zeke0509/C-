# ACM
- 包含头文件：`include <bits/stdc++.h>`

# 类型转换
- `string -> int`：`stoi(s);`
- `int -> string`：`to_string(i);`
- 遍历`string`得到就是`char`：
	```
	string s;
	for (char c : s) {}   // 正确
	```

# 字符串
- 截取字符串`string t = s.substr(pos, len);  // 从pos开始截取长度为len的字符串，不修改s`

# 数组
- 数组声明一定要用编译器常量
```
int lenS = s.size();
int rec[lenS] = {0};   // 错误，因为lenS不是编译器常量
vector<int> rec(lenS, 0);   // 正确
```
# `vector`
- 为已声明的`vector`预留长度：`vector.reserve(len)`

# 集合
- 插入元素：`st.insert(x);`
- 删除元素：`st.erase(x);`

# 映射
- 插入键值对：`mp.insert({"apple", 3});   // 若键存在，则不插入`
- 可以不用初始化就定义键，它会默认值为0，如下：
	```
	unordered_map<int, int> mp;
	mp[3]++;     // mp[3]默认生成一个{3,0}，然后++就变成1了
	```

# 排序
- `ranges::sort(nums)`：升序排序

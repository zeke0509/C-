# 一、字符串可以天生作为栈
题目：[1047. 删除字符串中的所有相邻重复项](https://leetcode.cn/problems/remove-all-adjacent-duplicates-in-string/description/)
## 1. `string`类型特点
字符串可以当成“字符栈”，因为它支持**尾部插入、尾部删除、访问尾元素**。
（1）`s.push_back(ch)`：对应`st.push(ch)`
（2）`s.pop_back()`：对应`st.pop()`
（3）`s.back()`：对应`st.top()`
（4）`s.empty()`：对应`st.empty()`

# 三、优先级队列
## 1. 大顶堆or小顶堆
选取原则：利用优先级队列是弹出堆顶元素。例如，我们想要维护前`k`个高频元素，那么我们要弹出小的，所以堆顶元素是小的，所以用小顶堆。
## 2. `Compare`的本质
`compare`是用来定义优先级的。当`compare(a, b)`为`true`时，`b`的优先级高于`a`，因此`b`更靠近堆顶。例如，`less<int>`本质上时`return a < b`，当`a < b`时返回`true`，`b`的优先级更高，而`b`是较大的元素，因此较大的元素优先级更高，会位于堆顶，所以这是大顶堆。
## 3. 优先级队列提供的接口
优先级队列`q`是不支持遍历的，即`for (auto& it : q)`是不被允许的，这是因为优先级队列不是可迭代容器，没有`begin()`和`end()`。
优先级队列对外提供以下接口：
- `top()`
- `push()`
- `pop()`
- `empty()`
- `size()`

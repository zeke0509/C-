1. 适用题型：无重复+有序

# 哈希表
1. 解决问题：快速判断一个元素是否出现在集合里
分为如下三种哈希结构
## 一. 数组
1. 适用题型：
	- 键是整数，且范围较小（0~100）
	- 计数哈希：统计每个数字出现次数
	- 存在性哈希：判断某些整数是否出现过

## 二. 集合
1. 适用题型：
	一般优先考虑使用==无序、无重复==集合`unordered_set`
2. 在集合中查找并删除元素
	(1)基本代码
	```cpp
	for (int val : nums2) {
		if (st.find(val) != st.end()) {
			vec.push_back(val);
			st.erase(val);
		}
	}
	```
	(2)改进
	`tip`：`st.erase()`是由返回值的，删除成功返回1
	```cpp
	for (int val : nums2) {
		if (st.erase(val)) vec.push_back(val);
	}
	```
